# Bitcoin Market Price Predictor

## Mission
To build a robust machine learning model that predicts the closing price of Bitcoin based on a combination of macroeconomic indicators and market sentiment, and to serve these predictions through a Flutter application via a REST API.

## Data Description & Source
The dataset used for this project is the **"Bitcoin Market Analysis Dataset (2021-2025)"**. It is a rich, multi-dimensional dataset containing historical daily data of Bitcoin prices alongside traditional market indicators (S&P 500, Gold), fiat currency strength (US Dollar Index - DXY), network hash rate, and market psychology metrics (Fear & Greed Index, Google Trends). 

**Source**: https://www.kaggle.com/datasets/purnamaridzkynugraha/bitcoin-historical-data
